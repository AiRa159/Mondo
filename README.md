# Mondo
The web application was written in JavaScript using the ReactJS library

## Technologies
- JavaScript, HTML, CSS
- ReactJS
- IntelliJ IDEA

## Getting started on local machine
In IDE terminal
```
npm install
npm start
```
To start the server
1. open package.json
2. find "scripts" section
3. run "dev-server"
